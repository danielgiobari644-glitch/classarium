Getting Started

Welcome to Classarium! This guide walks you through setting up your school management environment from scratch. Follow these steps to go from sign-up to managing your first batch of students.

Quick Start

Run the following commands to get a feel for the Classarium CLI and connect your school instance in under a minute.

# Install the Classarium CLI
npm install -g @classarium/cli

# Authenticate with your account
classarium auth login

# Initialize your school workspace
classarium init --school "Greenfield Academy"

# Verify connection
classarium status
# ✓ Connected to Greenfield Academy (production)
# ✓ API key: clm_live_••••••••4f2a
# ✓ 0 students, 0 classes synced
1

Create Your Account

Visit the Classarium signup page and create your administrator account. You'll need a valid email address, your full name, and your school's name. An email verification link will be sent to confirm your identity.

POST /api/v1/auth/register
Content-Type: application/json

{
  "email": "admin@greenfield.edu",
  "full_name": "Dr. Sarah Mitchell",
  "school_name": "Greenfield Academy",
  "password": "••••••••••••"
}

# Response: 201 Created
{
  "user": { "id": "usr_9xK2mN", "role": "admin" },
  "message": "Verification email sent to admin@greenfield.edu"
}
2

Configure Your School

Once verified, log in and complete the school configuration wizard. Set your academic session, grading system (e.g., 5-point or letter grades), term structure, and school logo. These settings will be used across all modules including report cards and result processing.

PUT /api/v1/schools/greenfield-academy/config
Authorization: Bearer clm_live_abc123...

{
  "grading_system": "5_point_scale",
  "terms_per_session": 3,
  "current_session": "2025/2026",
  "school_motto": "Excellence in Knowledge",
  "timezone": "Africa/Lagos"
}

# Response: 200 OK
{
  "school": { "name": "Greenfield Academy", "configured": true },
  "grading_scale": ["A", "B", "C", "D", "F"]
}
3

Add Students

Import your student roster using the bulk upload feature. Prepare a CSV file with columns for name, email, class, and admission_number. You can also add students individually through the dashboard or via the API.

POST /api/v1/students/import
Authorization: Bearer clm_live_abc123...
Content-Type: multipart/form-data

# Attach CSV file: students_batch_1.csv
# Columns: admission_number, full_name, email, class, gender

# Response: 202 Accepted
{
  "job_id": "imp_qR7tYz",
  "status": "processing",
  "total_rows": 342,
  "estimated_time": "12s"
}

# Check import status
GET /api/v1/students/import/imp_qR7tYz

# Response: 200 OK
{
  "status": "completed",
  "imported": 340,
  "skipped": 2,
  "errors": ["Duplicate admission number: STU-0105"]
}
4

Start Managing

With your school configured and students imported, you're ready to start managing day-to-day operations. Navigate to the dashboard to track attendance, record scores, generate report cards, and manage staff — all from a single unified interface.

GET /api/v1/dashboard/summary
Authorization: Bearer clm_live_abc123...

# Response: 200 OK
{
  "total_students": 340,
  "total_classes": 12,
  "total_staff": 28,
  "attendance_today": { "present": 315, "absent": 25 },
  "pending_results": 3,
  "report_cards_ready": 0
}
